# IPL Ticket Booking Platform\n\nA full-stack, production-ready clone of BookMyShow for TATA IPL ticket bookings built using Next.js (App Router), Tailwind CSS, Supabase, and Zustand.\n\n## Tech Stack\n- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS\n- **State Management**: Zustand\n- **Database & Auth**: Supabase PostgreSQL & Realtime\n- **Icons**: Lucide React\n\n## Setup & Local Development\n\n1. **Install Dependencies**\n   ```bash\n   npm install\n   ```\n\n2. **Environment Variables**\n   Create a `.env.local` file in the root directory and add your keys from `.env.example`:\n   ```env\n   NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url\n   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key\n   ```\n\n3. **Database Setup**\n   Go to your Supabase dashboard and run the SQL commands found in `supabase/schema.sql`.\n\n4. **Run the Development Server**\n   ```bash\n   npm run dev\n   ```\n\n## Admin Panel\n- **URL**: `/admin`\n- **Credentials**: `admin` / `admin123`\n\n## Deployment to Vercel\n1. Push the repository to GitHub.\n2. Import the project in Vercel.\n3. Add the Supabase URL and Anon Key to Vercel Environment Variables.\n4. Click **Deploy**.\n
